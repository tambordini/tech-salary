# Tech Salary Visualization

An interactive web application for visualizing technology industry salary data in Thailand, built with Svelte + TypeScript.

## Features

- Salary comparison across different tech roles
- Experience level breakdown
- Company size analysis
- Location-based salary differences
- Technology stack correlation with compensation

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Start development server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```

## Tech Stack

- Svelte + TypeScript
- Vite
- D3.js for data visualization
- TailwindCSS for styling
